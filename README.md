# chandu-distance
stupid program to calculate distance (combine Manhattan and euclidean but isnt Minkowski )

# use case
in grid based 2d game you may need to calculate distance b/w 2 points where player can move only in 1 or 2 axis (x,y or both) at once 
or in 3d games player can also move diagonaly on stairs(wide stairs, player move in 3 axis at once, including z axis)
